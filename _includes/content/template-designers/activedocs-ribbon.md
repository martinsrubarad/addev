The ActiveDocs Ribbon is added to Microsoft Word when ActiveDocs is installed. It provides access to all design tools for creating and managing Template automation.

## Accessing the ActiveDocs Ribbon

The ActiveDocs Ribbon tab appears in the Microsoft Word ribbon when a Template (.dotm) is opened from ActiveDocs WorkCenter Client. Select the **ActiveDocs** tab to display the available design tools.

> **TIP:** You can use shortcut keystrokes in the ActiveDocs Ribbon: press **Alt** to display key tips, then press the corresponding letter or number to activate a tool.

## Design Tools

The following table lists all tools available on the ActiveDocs Ribbon.

| Design Tool | Definition |
| --- | --- |
| WorkCenter Client | Opens ActiveDocs WorkCenter Client. |
| Design Items | Opens the Design Items Dialog for managing Groups, Active Fields, Repeating Item Groups, Charts, Snippet Links, and Rules. |
| Designer Pane | Toggles the Designer Pane on or off. The Designer Pane provides a visual overview of all markers in the Template. |
| Snippet Pane | Toggles the Snippet Pane on or off. The Snippet Pane allows browsing and inserting Snippets. |
| Insert Group Marker | Inserts a Group marker at the current cursor position in the Template. |
| Insert Active Field Marker | Inserts an Active Field marker at the current cursor position in the Template. |
| Insert Rule Start Marker | Inserts a Rule Start marker at the current cursor position in the Template. |
| Insert Rule End Marker | Inserts a Rule End marker at the current cursor position in the Template. |
| Insert Repeating Item Group Start Marker | Inserts a Repeating Item Group Start marker at the current cursor position. |
| Insert Repeating Item Group End Marker | Inserts a Repeating Item Group End marker at the current cursor position. |
| Insert Snippet Link Marker | Inserts a Snippet Link marker at the current cursor position. |
| Insert Chart Marker | Inserts a Chart marker at the current cursor position. |
| Select Marker | Selects the ActiveDocs marker nearest to the current cursor position. |
| Go to Marker | Opens the Go to Marker dialog for navigating to a specific marker in the Template. |
| Marker Properties | Displays the properties of the currently selected marker. |
| Convert Marker to Text | Converts the selected ActiveDocs marker to plain text, removing its automation. |
| Insert Paragraph Before | Inserts a new paragraph before the currently selected marker. |
| Insert Paragraph After | Inserts a new paragraph after the currently selected marker. |
| Test Template | Launches the Document Wizard in Designer Test Mode for the current Template. |
| Refresh | Refreshes the Template to reflect any external changes (e.g., updated Design Item Sets or Snippets). |
| Save | Opens the ActiveDocs Save dialog to save the Template back to the WorkCenter Site. |
| Help | Opens ActiveDocs Help documentation. |

> **TIP:** The most frequently used tools during Template design are **Design Items**, **Designer Pane**, **Insert Active Field Marker**, **Insert Rule Start Marker**, **Insert Rule End Marker**, and **Test Template**. Familiarize yourself with these first.
