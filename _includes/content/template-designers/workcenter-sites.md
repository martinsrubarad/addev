Within the WorkCenter Client, Design Components are stored collectively in repositories called WorkCenter Sites. The WorkCenter Client provides the ability to connect to multiple WorkCenter Sites for storing and managing your Design Components.

All WorkCenter Sites have the same types of Design Components. The operations you can perform on a WorkCenter Site may be limited by the permissions set for you by the administrator.

## Adding a WorkCenter Site Connection

1. On the **WorkCenter Sites** Panel, select **Add WorkCenter Site**.
2. The **Add ActiveDocs WorkCenter Site Connection** dialog appears:
   - **Name** — Enter the name of the WorkCenter Site. This is the name displayed in the connection list.
   - **WorkCenter URL** — Enter the WorkCenter Site URL. Contact your ActiveDocs Administrator if you do not know the URL.
   - **Default WorkCenter Site** — Select to set this as your default WorkCenter Site.
3. Click **OK**.
4. If the connection requires more than your usual Windows login, the **Windows Security** dialog appears:
   - **User Name** — Enter your User name.
   - **Password** — Enter your Password.
5. Click **OK**.

> **NOTE:** The first WorkCenter Site you connect to will automatically become the Default WorkCenter Site. If you connect to more than one, you can set your Default by editing the WorkCenter Site properties.

## Adding a Recent WorkCenter Site Connection

1. On the WorkCenter Sites Panel, select **Recent WorkCenter Sites**. The Recent WorkCenter Sites will be listed on the right-hand side.
2. Select the WorkCenter Site to connect to. The **Connection Properties** dialog appears.
3. View information, make any desired edits, and click **OK**.

## Removing a WorkCenter Site Connection

1. On the WorkCenter Sites Panel, select the WorkCenter Site to close from the connection list.
2. Select **Remove this WorkCenter Site Connection**.
3. The confirmation dialog appears. Click **Yes** to confirm removal.

## Viewing or Editing Connection Properties

1. On the WorkCenter Sites Panel, select the WorkCenter Site from the connection list.
2. Select **Edit WorkCenter Site Connection**. The **Connection Properties** dialog is displayed.
3. View or make the desired edits and click **OK**.

## Creating a WorkCenter Site Shortcut

1. On the WorkCenter Sites Panel, select the WorkCenter Site from the connection list.
2. Select **Create WorkCenter Site shortcut**.
3. Click **OK** in the confirmation dialog.

The shortcut will be saved to your desktop and will automatically connect to the WorkCenter Site.

> **NOTE:** You can have multiple WorkCenter Client sessions open at the same time using your shortcuts.

## Browsing to a WorkCenter Site

1. On the WorkCenter Sites Panel, select the WorkCenter Site from the connection list.
2. Select **Browse to this WorkCenter Site**. The WorkCenter screen is displayed in your browser.
